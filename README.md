# 学生信息管理系统 python534

## 项目概述

学生信息管理系统是一款基于Python和MySQL开发的应用，主要用于处理和管理学生的基本信息、课程选课以及成绩等。该系统提供了简洁、直观的操作界面，并通过权限控制实现对管理员和学生的区分管理。

## 技术栈

- 后端：Python
- 数据库：MySQL
- 开发工具：PyCharm

## 功能模块

### 首页功能

- 管理员登录
- 增删学生信息
- 选课与退课操作
- 增删改查课程信息

## 系统角色

- 管理员：负责登录系统，管理学生信息及课程

## 运行环境

- Python环境
- MySQL数据库环境

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库
3. 导入数据库结构
4. 运行系统
